# 好运电商 - 用户使用指南

## 一、注册与登录

### 注册流程
1. 打开好运电商 APP 或官网
2. 点击"注册"，输入手机号
3. 获取并输入短信验证码
4. 设置登录密码（8-20位，需包含字母和数字）
5. 注册成功，自动登录

### 第三方登录
支持微信、QQ、支付宝快捷登录，首次登录需绑定手机号。

### 忘记密码
1. 登录页点击"忘记密码"
2. 输入注册手机号
3. 通过短信验证码验证身份
4. 设置新密码

## 二、商品浏览与搜索

### 搜索技巧
- 支持关键词搜索（商品名称、品牌、型号）
- 支持按分类筛选（服装、数码、家居、食品等）
- 支持按价格区间、销量、评价排序
- 搜索栏可输入语音搜索

### 商品详情页信息
- 商品图片（支持放大查看）
- 价格信息（原价、促销价、会员价）
- 规格选择（颜色、尺码等）
- 商品描述、参数详情
- 用户评价和晒单

## 三、下单与支付

### 下单流程
1. 选择商品规格，点击"立即购买"或"加入购物车"
2. 确认收货地址（支持新增/修改地址）
3. 选择配送方式（普通快递/次日达/自提）
4. 使用优惠券或积分抵扣（可选）
5. 确认订单信息，提交订单
6. 在30分钟内完成支付，超时订单自动取消

### 支持的支付方式
- 支付宝
- 微信支付
- 银行卡支付（储蓄卡/信用卡）
- 花呗分期（部分商品支持3/6/12期免息）
- 好运余额

### 支付失败怎么办？
1. 检查网络连接是否正常
2. 确认账户余额是否充足
3. 尝试更换支付方式
4. 检查银行卡是否限额
5. 联系银行客服确认支付状态

## 四、物流与配送

### 配送时间
- 普通快递：1-3个工作日
- 次日达：当日18:00前下单，次日送达
- 自提：下单后1-2个工作日到自提点

### 物流查询
1. 打开APP → "我的订单"
2. 点击对应订单查看物流详情
3. 支持复制运单号到快递官网查询
4. 物流状态实时更新（已发货、运输中、派送中、已签收）

### 签收注意事项
- 收到包裹先验货再签收
- 如有破损或数量不对，可拒签
- 签收后发现问题请在48小时内联系客服

## 五、退换货政策

### 7天无理由退换货
- 商品未使用、不影响二次销售
- 服装类需吊牌完好
- 数码产品激活后不支持无理由退货
- 食品、定制商品不支持无理由退货

### 退货流程
1. 打开APP → "我的订单" → 申请退货
2. 选择退货原因，上传照片（如有问题）
3. 等待卖家审核（24小时内）
4. 审核通过后按提示寄回商品
5. 卖家确认收货后退款到原支付账户

### 退款到账时间
- 原路退回：1-3个工作日
- 退到好运余额：即时到账
- 花呗/信用卡分期：1-5个工作日

## 六、会员体系

### 会员等级
| 等级 | 条件 | 专属权益 |
|------|------|----------|
| 普通会员 | 注册即可 | 基础权益 |
| 银卡会员 | 年消费≥500元 | 95折优惠、生日礼包 |
| 金卡会员 | 年消费≥2000元 | 9折优惠、免运费、专属客服 |
| 钻石会员 | 年消费≥5000元 | 85折优惠、优先发货、新品试用 |

### 积分规则
- 每消费1元获得1积分
- 签到每天获得5积分
- 评价商品获得10积分
- 100积分 = 1元，下单时可抵扣
- 积分有效期为获得后1年

### 优惠券
- 新用户注册送10元无门槛券
- 每月会员日发放专属券
- 满99减10、满199减30等满减券
- 优惠券不可叠加使用
- 优惠券有效期一般为7-30天

## 七、账户安全

### 账户保护建议
- 不要将密码告知他人
- 定期修改密码
- 绑定手机号用于找回密码
- 开启支付验证（指纹/面部识别）

### 账户被盗怎么办？
1. 立即修改密码
2. 联系客服冻结账户
3. 检查近期订单和支付记录
4. 如有损失，提交工单申请处理
