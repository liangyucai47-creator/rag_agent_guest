# 速游加速器 - 使用指南

## 快速开始

### 下载与安装
1. 前往速游加速器官网下载最新版本客户端
2. 双击安装包，按提示完成安装
3. 首次打开需要注册/登录账号

### 账号注册
- 支持手机号注册、微信登录、QQ登录
- 新用户可免费试用3天VIP加速
- 一个手机号仅可注册一个账号

## 使用方法

### 添加游戏
1. 打开速游客户端，点击"添加游戏"
2. 在搜索框输入游戏名称（支持中英文）
3. 选择对应的游戏版本（国服/国际服/Steam）
4. 点击"添加"完成

### 开始加速
1. 在游戏列表中选择要加速的游戏
2. 点击"一键加速"按钮
3. 等待加速连接成功（显示绿色延迟）
4. 启动游戏即可享受加速服务

### 选择节点
- 系统默认自动选择最优节点
- 也可以手动切换节点
- 建议选择延迟最低的节点（绿色 < 50ms）
- 延迟 > 150ms 建议切换其他节点

## 常见问题

### 加速失败怎么办？
1. 检查网络连接是否正常
2. 尝试切换其他节点
3. 关闭其他可能冲突的加速器/VPN
4. 以管理员身份运行速游客户端
5. 检查防火墙是否拦截
6. 重启客户端后再试
7. 如果以上都无法解决，请联系客服提交工单

### 加速后延迟仍然很高？
1. 尝试切换不同线路（电信/联通/移动）
2. 选择距离更近的节点
3. 关闭后台占用带宽的程序
4. 检查本地网络是否稳定
5. 联系客服反馈节点问题

### 加速后游戏进不去？
1. 确认游戏版本是否为最新
2. 检查游戏路径是否正确
3. 尝试关闭加速后启动游戏，再开启加速
4. 某些游戏需要关闭加速后更新

### 支持哪些游戏？
- 支持市面上绝大多数热门游戏
- 包括 Steam、Epic、Origin、Uplay 等平台游戏
- 国内外手游、端游、主机云游戏
- 具体支持列表可在客户端内搜索查看
- 如果找不到游戏，可以通过反馈功能提交

## 技术支持

### 系统要求
- Windows 7 及以上

- Android 5.0 及以上
- iOS 10.0 及以上

### 常见错误码
- E1001: 网络连接超时，请检查网络
- E1002: 节点不可用，请切换节点
- E1003: 账号异常，请联系客服
- E1004: 加速冲突，请关闭其他加速工具
- E1005: 版本过低，请更新客户端
